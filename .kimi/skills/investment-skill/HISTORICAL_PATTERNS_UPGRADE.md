### 🤖 智能匹配算法

**新功能**:
- 加权指标匹配
- 匹配度评分 (0-100%)
- 多模式同时匹配
- 详细匹配报告

**匹配逻辑**:
```python
# 每个历史模式包含指标阈值和权重
pattern = {
    "us_jp_spread_2y": {"threshold": 3.5, "weight": 3.0},
    "dxy": {"threshold": 105, "weight": 2.5},
    "vix": {"threshold": 25, "weight": 2.0}
}

# 当前指标触发 → 获得对应权重
# 匹配度 = 已触发权重 / 总权重

# 示例:
# 如果美日利差3.2% (critical) 触发 → 获得3.0分
# 如果DXY 106 (elevated) 触发 → 获得2.5分
# 总权重假设10分 → 匹配度55%
```

### 案例1: 2024年8月日元套利unwind
**文件**: `2024_08_yen_carry_trade_unwind.md`

**关键指标**:
- 美日利差 < 3.5%
- USDJPY单日下跌 > 1.5%
- 日债收益率跳涨 > 20bp
- VIX > 25

**市场影响**:
- 日经225: -12.4%
- 纳斯达克: -8% (3天)
- 半导体: -15%

**理想操作**:
- T-48小时: 减仓50%科技股
- T-24小时: 再减仓30%
- 结果: 避免48万损失(基于600万组合)

**经验教训**:
- 美日利差<3.5% + 日元升值 = 危机信号
- 不要抄底，等待VIX回落
- 黄金对冲有效

## 💡 核心价值

### 避免"这次不一样"陷阱

**没有历史模式匹配**:
> "市场跌了，但这次不一样，是XX原因"
> → 盲目持有 → 大亏

**有历史模式匹配**:
> "当前指标组合匹配2024年8月模式(75%)"
> "历史上类似情况导致-10%跌幅"
> → 提前减仓 → 避免亏损

### 量化风险

**匹配度评分**:
- 0-30%: 不匹配，正常交易
- 30-50%: 轻度匹配，保持警惕
- 50-70%: 中度匹配，准备行动
- 70%+: 高度匹配，立即行动
