# 领先指标预警系统 (Leading Indicator Alert System)

## 🎯 核心理念：从"事后提醒"到"事前预警"

### ❌ 传统预警（滞后性）
```
"你的基金跌了3%，请注意"
↓
问题：价格已经跌了，来不及应对
```

### ✅ 领先指标预警（前瞻性）
```
"监测到流动性收紧信号，48小时内可能下跌"
↓
优势：提前减仓，避免亏损
```

---

## 🔬 监控的领先指标（先行信号）

### 1. 日元套利交易信号 (JPY Carry Trade)
**指标**: 日债收益率、USDJPY汇率
**逻辑**: 日债收益率跳涨 → 日元升值 → 套利交易 unwind → 全球流动性收紧
**历史案例**: 2024年8月日元套利交易 unwind 引发全球市场-10%
**提前时间**: 1-3天

### 2. 美日利差 (US2Y-JP2Y Spread)
**指标**: 美债2年 - 日债2年收益率差
**逻辑**: 利差收窄 → 套利交易吸引力下降 → 资金回流日本
**危险阈值**: < 3.0% (Critical), < 4.0% (Elevated)
**提前时间**: 1-7天

### 3. MOVE指数/VIX (美债波动率)
**指标**: 美债市场恐慌指数
**逻辑**: MOVE > 120 → 美债流动性枯竭 → 风险资产抛售
**危险阈值**: VIX > 25 (Elevated), > 30 (Critical)
**提前时间**: immediate (即时)

### 4. 美元指数DXY (全球美元流动性)
**指标**: 美元对一篮子货币汇率
**逻辑**: DXY > 105 → 强势美元 → 新兴市场资金外流
**危险阈值**: > 105 (Elevated), > 107 (Critical)
**提前时间**: 1-4周

### 5. 收益率曲线 (Recession Signal)
**指标**: 10年期 - 2年期美债利差
**逻辑**: 倒挂 → 衰退预警 → 企业盈利下滑
**危险信号**: 负值（倒挂）
**提前时间**: 6-18个月

### 6. VIX期限结构 (恐慌预期)
**指标**: 近月vs远月波动率
**逻辑**: 近月 > 远月 → 即期恐慌 → 市场压力

```bash

```

### 持续监控（推荐）
```bash
# 每小时检查一次
uv run python skills/leading_indicator_alerts/__init__.py --monitor

# 每4小时检查
uv run python skills/leading_indicator_alerts/__init__.py --monitor --interval 14400
```

### 添加到run.sh
编辑 `~/investment-skill/run.sh`，添加:
```bash
leading|li)
    echo "🔮 Running Leading Indicator Analysis..."
    uv run python skills/leading_indicator_alerts/__init__.py $@
    ;;
```

然后使用: `./run.sh leading`

## 🔄 两种预警系统的对比

| 特性 | 传统预警 (Price-based) | 领先指标预警 (Leading Indicator) |
|------|------------------------|----------------------------------|
| **监控对象** | 价格、涨跌幅 | 流动性、利差、波动率 |
| **时间性质** | 滞后 (已经发生了) | 前瞻 (将要发生) |
| **预警时间** | 事后提醒 | 提前24-72小时 |
| **输出** | "跌了3%" | "流动性紧缩，建议减仓" |
| **可行动性** | 无法行动(已发生) | 可行动(提前应对) |
| **避免回撤** | ❌ 不能 | ✅ 能 |
| **复杂度** | 简单 | 需要专业知识 |

## ⚠️ 局限性

1. **假阳性**: 领先指标有时会发出错误信号
2. **时间不确定**: "1-3天"是估计，实际可能更短或更长
3. **黑天鹅**: 无法预测完全意外的事件
4. **数据延迟**: 部分宏观数据有滞后
