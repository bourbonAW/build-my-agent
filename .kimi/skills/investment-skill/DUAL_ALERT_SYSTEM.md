# 投资Agent - 双预警系统对比

## 🎯 你现在拥有两套预警系统

### 1️⃣ 传统价格预警（Price-Based）
**监控**: 已经发生的价格变动  
**用途**: 盘后复盘、止损提醒  
**时效**: 事后（滞后）  

**使用**:
```bash
# 查看已发生的预警
./run.sh fund-monitor --alert

# 实时监控价格变动
uv run python skills/realtime_alerts/__init__.py --monitor
```

**示例输出**:
```
🟡 [FUND] 国泰有色矿业ETF - Significant Decline
   Daily decline of -3.33% exceeds threshold
   含义：已经跌了3.33%，请注意
```

---

### 2️⃣ 领先指标预警（Leading Indicator）⭐ 推荐
**监控**: 预示未来价格变动的信号  
**用途**: 提前减仓/对冲，避免亏损  
**时效**: 前瞻（提前24-72小时）  

**使用**:
```bash
# 立即分析当前风险
uv run python skills/leading_indicator_alerts/__init__.py

# 持续监控（每小时检查）
uv run python skills/leading_indicator_alerts/__init__.py --monitor

# 高频监控（重大事件期间）
uv run python skills/leading_indicator_alerts/__init__.py --monitor --interval 1800
```

**示例输出**:
```
🔴 Warning Level: RED
⚠️ 严重流动性紧缩风险！2个关键指标触发危机级别

关键信号:
🚨 美日利差收窄至3.2% (Critical)
🚨 日元套利交易压力指标触发 (Critical)  

历史模式匹配: 2024年8月日元套利交易 unwind 模式

战略建议:
📉 减仓 (immediate) - 立即降低高beta持仓，保护本金
🛡️ 对冲 (this_week) - 增配黄金对冲风险

前瞻性总结:
🚨 紧急预警！领先指标显示市场面临重大调整风险，
历史模式匹配显示类似情况通常导致快速下跌，时间窗口1-3天。
```

---

## 📊 两套系统的对比

| 维度 | 价格预警 | 领先指标预警 |
|------|----------|--------------|
| **核心问题** | "跌了多少？" | "将要发生什么？" |
| **监控对象** | 基金价格、涨跌幅 | 利差、波动率、流动性 |
| **时间方向** | 向后看（滞后） | 向前看（领先） |
| **行动价值** | ❌ 无法避免亏损 | ✅ 提前避险 |
| **复杂度** | 简单 | 需要专业理解 |
| **预警时效** | 已发生 | 提前1-3天 |
| **避免回撤** | 不能 | 能 |

## 🎓 使用建议

### 日常流程（推荐）

**每天早上 8:30**（开盘前）:
```bash
# 1. 检查领先指标（决定今日策略）
uv run python skills/leading_indicator_alerts/__init__.py
#    ↓
#    如果是 RED/ORANGE → 考虑减仓
#    如果是 GREEN → 正常交易

# 2. 检查持仓状态（了解当前盈亏）
./run.sh fund-monitor
```

**盘中**（重大事件日）:
```bash
# 开启领先指标监控
uv run python skills/leading_indicator_alerts/__init__.py --monitor --interval 3600
```

**收盘后**:
```bash
# 生成完整日报（包含价格预警）
./run.sh
```

## 🔄 实际案例：2024年8月的教训

### 如果只使用价格预警
- 8月5日：收到预警"纳斯达克跌了-5%"
- 问题：已经跌了，无法避免
- 结果：组合亏损-10%

### 如果使用领先指标预警
- 8月3日（周五）：系统发出RED警告
  - 日债收益率跳涨
  - 美日利差收窄
  - 匹配"2024年8月套利unwind"模式
  - 建议：减仓科技股QDII
  
- 8月4-5日（周末-周一）：市场暴跌
  - 因为你提前减仓，避免了-10%回撤
  - 同时增配黄金对冲，黄金+5%
  
- 结果：总组合只亏-2%（vs -10%）

**差异**: 8%的回撤避免 = 48万的保护（基于你的600万组合）
